# Felipa.io landing page

Bespoke single-page site prepared for Analys Falchuk and the Felipa.io team ahead of eMerge Americas 2026 (booth SS20). Carries the pitch body from `fit-analysis.md` section 7, wrapped in Felipa's own brand system: lime (#c7f500) on deep navy (#08052b), Lexend throughout, warm and plainspoken register. No JavaScript. No Eduba branding at the top; Eduba appears only as the small footer credit and inside named case study references.

Intended URL path: /for/felipa-io

- `index.html` — page markup, meta description pulled from the booth hook
- `styles.css` — brand tokens, responsive down to 375px
- `logo.png` — Felipa.io logo, copied from `../Brand Pack/assets/logo.png`
